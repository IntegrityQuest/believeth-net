# believeth.net

A living expression of the [Cross-Cultural AI Integrity Charter](https://github.com/IntegrityQuest/cross-cultural-ai-integrity-charter) and the Golden AI Rule.

---

## What This Is

believeth.net is a Christian unity project built on the IQ framework — an attempt to find the shared root beneath 40,000 separate Christian paths, using the Golden Rule as the common ground.

If Christians can find unity in the principle they already share, it becomes a proof of concept for the wider human family.

---

## Foundation

This project is built on the Cross-Cultural AI Integrity Charter:

> *Ethical AI does not require machines that love humanity. It requires systems that are incapable of betraying it.*

The charter documents that the Golden Rule — treat others as you wish to be treated — appears independently across 50 wisdom traditions spanning the full breadth of human history. These traditions never met. They reached the same conclusion.

believeth.net applies that convergent wisdom to Christian unity.

---

## The Vision

- Unite Christians around the Golden Rule as shared foundation
- Use AI to bridge language and denominational division
- Build content that children can carry across cultural barriers
- Demonstrate that unity is possible before reaching for the wider human family
